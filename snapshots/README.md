# Snapshots

Timestamped visual evidence of how Tone Canvas evolves — for authorship record and design
history.

## What goes here

- **Screenshots** of the canvas / specific tone treatments (`.png`, `.jpg`).
- **SVG outputs** exported from the prototype (`.svg`).
- **Short screen recordings** of interactions (`.gif`, `.mp4`, kept small).

## Naming

Prefix every file with an ISO date so the folder sorts chronologically:

```
YYYY-MM-DD-short-description.ext
```

Examples:

```
2026-06-26-stitched-third-tone.png
2026-06-26-connected-wave-sentence.svg
2026-06-27-marquee-multiselect.gif
```

## Logging snapshots

For each snapshot, add a one-line note below describing **what changed** or what it
demonstrates. Keep newest at the top.

| Date | File | What it shows |

> Tip: capture a snapshot at every meaningful milestone (new tone treatment, new
> interaction, export format) so the visual lineage stays dated and continuous.
